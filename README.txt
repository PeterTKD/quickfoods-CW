Link to live version httpsvisionary-crisp-6ba2fe.netlify.app


The main functionalities of the website are the following 
-	Search recipes 
-	Add a recipe to chart 
-	Add recipe to menu
-	Like and dislike a recipe
-	Search ingredient nutrition value 
-	Menu page with the instructions for cooking selected meal
-	Shopping list with the ingredients for each recipe

To Run the application
npm install
npm start
